# python_para_dados

Curso de Python para Análise de Dados, onde desenvolvi um projeto prático utilizando dados públicos do Portal da Transparência, com foco na análise de viagens a serviço em 2023.

Ao longo do curso, trabalhei com um grande volume de dados (mais de 800 mil registros), aplicando técnicas essenciais de limpeza, transformação, análise e visualização de dados utilizando Python e Pandas.

 Principais atividades desenvolvidas:

Importação e tratamento de dados reais em CSV

Limpeza de dados (valores nulos, padronização de textos e conversão de tipos)

Manipulação de strings e séries booleanas

Criação de novas métricas e colunas (como despesas totais e duração das viagens)

Análises estatísticas por cargo público (médias, totais e proporções)

Agrupamentos e consolidação de dados

Integração de bases (merge entre viagens e passagens)

Criação e personalização de gráficos com Matplotlib

Exportação dos resultados para Excel e geração de visualizações finais

O projeto resultou em insights relevantes sobre gastos públicos, frequência de viagens por cargo e comportamento das despesas, reforçando a importância da análise de dados para transparência, tomada de decisão e controle de recursos.

Esse aprendizado fortaleceu ainda mais minhas habilidades em Python para Análise de Dados, além de consolidar conceitos fundamentais de Data Analytics aplicados a problemas reais.
